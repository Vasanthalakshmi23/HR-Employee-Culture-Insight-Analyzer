# Multi-Modal HR Employee Culture Insight Analyzer

## 📌 Overview
The **Multi-Modal HR Employee Culture Insight Analyzer** is an AI-powered analytics platform developed to evaluate and improve workplace culture using multiple forms of employee data such as event images, textual feedback, and performance metrics.

The system combines **Computer Vision**, **Natural Language Processing (NLP)**, and **Machine Learning** techniques to generate meaningful organizational insights. It helps HR teams understand employee engagement, participation levels, emotional responses, and workplace satisfaction through intelligent data analysis.

This project aims to support organizations in making data-driven decisions that enhance employee well-being, collaboration, and overall productivity.

---

# 🎯 Objectives

- Analyze employee participation and emotions from workplace event images
- Extract employee sentiments and opinions from feedback text
- Generate workplace culture insights and engagement scores
- Visualize HR analytics through interactive dashboards

# 📂 Project Modules

### 1. Event Image Activity Analyzer
Analyzes employee event images to detect participation, engagement, and emotional expressions using computer vision techniques.

### 2. NLP Employee Feedback Extractor
Processes employee feedback and survey responses to identify sentiments, workplace concerns, and employee opinions.

### 3. Culture Score Model
Generates an overall workplace culture score based on employee engagement, feedback analysis, and participation metrics.

### 4. HR Visualization Dashboard
Displays organizational analytics, engagement reports, and workplace culture trends using interactive visualizations.

### 5. Employee User Interface
Provides an easy-to-use interface for employees and HR teams to interact with the system and access reports.
